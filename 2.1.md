Кейс 2.
На каких уровнях системы работают следующие службы или приложения?
оконный менеджер;
файловый менеджер;
веб-браузер;
текстовый редактор;
калькулятор.

ответ:

оконный менеджер - o/s service

файловый менеджер - user applications

веб-браузер - user applications
